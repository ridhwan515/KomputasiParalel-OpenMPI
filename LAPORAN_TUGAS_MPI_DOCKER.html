2. Penjelasan Script

Program utama yang digunakan pada direktori laporan ini adalah mpi_nilai2.c. Program membaca file data_nilai.csv, mengambil kolom Nilai, memvalidasi nilai dalam rentang 0 sampai 100, lalu menghitung nilai maksimum, nilai minimum, dan rata-rata kelas.

Konstanta dan File Input

Bagian konfigurasi awal menentukan nama file CSV bawaan, kapasitas awal array, dan ukuran buffer pembacaan baris. File default yang dibaca adalah data_nilai.csv.

#define DEFAULT_INPUT_FILE "data_nilai.csv"
#define INITIAL_CAPACITY 128
#define LINE_SIZE 256

Pembacaan CSV

Fungsi read_scores membuka file CSV, melewati header, mencari tanda koma pada setiap baris, lalu mengubah isi kolom setelah koma menjadi angka bertipe double. Baris yang tidak valid atau nilai yang berada di luar rentang 0 sampai 100 diabaikan.

fgets(line, sizeof(line), file);

while (fgets(line, sizeof(line), file) != NULL) {
    char *comma = strchr(line, ',');
    score = strtod(comma + 1, &endptr);
    scores[count] = score;
    count++;
}

Perhitungan Nilai

Setelah semua nilai berhasil dimuat ke memori, program melakukan iterasi terhadap array all_scores. Pada setiap iterasi, program memperbarui nilai maksimum, nilai minimum, dan total nilai. Rata-rata dihitung dengan membagi total nilai dengan jumlah data yang valid.

for (int i = 0; i < total_count; i++) {
    double score = all_scores[i];

    if (score > max_score) {
        max_score = score;
    }
    if (score < min_score) {
        min_score = score;
    }
    sum_score += score;
}

double average = sum_score / (double)total_count;

Output Akhir

printf("===========================================\n");
printf("HASIL ANALISIS NILAI MAHASISWA (SEKUENSIAL)\n");
printf("===========================================\n");
printf("Total Data Diproses : %d baris\n", total_count);
printf("Nilai Tertinggi     : %.2f\n", max_score);
printf("Nilai Terendah      : %.2f\n", min_score);
printf("Rata-rata Kelas     : %.2f\n", average);
printf("===========================================\n");

Output dibuat mengikuti format tugas agar hasil akhir mudah dibandingkan dengan data CSV yang digunakan.

3. Konfigurasi Lingkungan

Lingkungan pengerjaan berada pada VPS dosen dengan alamat login:

ssh kelasa@103.253.213.240

Di dalam VPS tersebut, Docker digunakan untuk menyediakan lingkungan eksekusi yang sama untuk seluruh anggota kelompok. Dengan demikian, kelompok tidak perlu lagi menghubungkan laptop atau node anggota satu per satu menggunakan jaringan lokal.

Komponen Peran Hostname Slot
Container 1 Master kelas-a-master 2
Container 2 Worker kelas-a-worker1 2
Container 3 Worker kelas-a-worker2 2

Isi file hosts.txt yang ada pada direktori laporan:

kelas-a-master slots=2
kelas-a-worker1 slots=2
kelas-a-worker2 slots=2

Kendala yang ditangani:

  • Tidak perlu menyamakan jaringan antar laptop anggota karena semua proses dilakukan di satu VPS.
  • Docker membuat nama host container dapat dipanggil langsung melalui hostname, misalnya kelas-a-master dan kelas-a-worker1.
  • File hosts.txt menggunakan format hostname dan jumlah slot, bukan IP laptop anggota.
  • Dataset data_nilai.csv dan source mpi_nilai2.c ditempatkan pada direktori kerja yang sama agar program dapat membaca input tanpa path tambahan.

5. Hasil Akhir dan Analisis

Command masuk ke VPS dosen:

ssh kelasa@103.253.213.240

Command masuk ke container Docker utama:

docker exec -it kelas-a-master bash

Command compile di dalam container:

gcc mpi_nilai2.c -o mpi_nilai2

Command eksekusi di dalam container Docker:

mpirun --allow-run-as-root --hostfile -np 6 ./mpi_nilai2

Output program:

===========================================
HASIL ANALISIS NILAI MAHASISWA (SEKUENSIAL)
===========================================
Total Data Diproses : 100 baris
Nilai Tertinggi     : 98.50
Nilai Terendah      : 62.40
Rata-rata Kelas     : 80.94

Analisis:

  • File data_nilai.csv memiliki 101 baris, terdiri dari 1 baris header dan 100 baris data nilai mahasiswa.
  • Nilai tertinggi dari dataset adalah 98.50.
  • Nilai terendah dari dataset adalah 62.40.
  • Rata-rata kelas dari 100 data valid adalah 80.94.
  • Alur baru lebih sederhana karena seluruh eksekusi berada di VPS dan container Docker, sehingga tidak memerlukan koneksi antar laptop anggota.
